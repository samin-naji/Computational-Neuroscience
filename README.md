# 🧠 Computational Neuroscience Projects

## Overview
This repository contains my exercises and projects for a computational neuroscience course.  
The work spans various topics, including machine learning, deep learning, data visualization, and EEG signal processing.

### 📦 Core Python Libraries
- `os`

### 📊 Data Handling & Computation
- `pandas`
- `numpy.fft`
- `scipy.io.loadmat`

  - `spectrogram`, `signal`, `stft`, `welch`, `hann`, `butter`, `filtfilt`
